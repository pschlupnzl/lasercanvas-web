Interface

In this chapter, we introduce the elements of the LaserCanvas user interface. The most important parts are the Property Manager and the Canvas where the systems are edited. In addition, a number of other tools, such as graphs and an inventory list, are useful for designing laser systems. These design tools can be accessed under the Tools menu.

The LaserCanvas design tools:

  • Property Manager
  • Canvas
  • Mode Plot
  • System Graph
  • Optic Graph
  • Inventory
  • Solver
  • Wireframe
  • Saved Files
  • Printing

Each system has at most one Canvas, Inventory, Solver, and Wireframe windows, but any number of Mode Plot, System Graph, and Optic Graph windows.
