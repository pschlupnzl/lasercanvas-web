Data Files

LaserCanvas saves data files in a text format. Although not directly intended to be edited by the user, the file format is described in this chapter for reference.

WARNING
Creating your own data files and loading them into LaserCanvas can lead to unpredictable results.
You must ensure that your systems are physically meaningful.
